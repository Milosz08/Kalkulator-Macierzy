\doxysubsection{Opis szczegółowy}
Klasa pochodna dziedzicząca po klasie abstrakcyjnej \char`\"{}\+Matrix\+Abstract\char`\"{}. Obsługuje macierze diagonalne. Macierze diagonalne to specjalny typ macierzy kwadratowej, która poza swoją główną przekątną -\/ diagonalną jest wypełniona zerami. 

Klasa pochodna dziedzicząca po klasie abstrakcyjnej \char`\"{}\+Matrix\+Abstract\char`\"{}. Obsługuje zwykłe macierze prostokątne oraz kwadratowe.
